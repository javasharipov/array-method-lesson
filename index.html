Master JavaScript Array Methods

Explore 12 essential JavaScript array methods. These methods are grouped as **mutable**, **immutable**, and **other methods** to enhance your understanding of how they manipulate arrays.

Mutable Methods

.push()

Adds elements to the end of an array.

let arr = [1, 2];
arr.push(3);
console.log(arr); // [1, 2, 3]

.pop()

Removes the last element from an array.

let arr = [1, 2, 3];
arr.pop();
console.log(arr); // [1, 2]

.splice()

Removes or replaces elements at a specific index.

let arr = [1, 2, 3];
arr.splice(1, 1, 99);
console.log(arr); // [1, 99, 3]

.shift()

Removes the first element from an array.

let arr = [1, 2, 3];
arr.shift();
console.log(arr); // [2, 3]

.unshift()

Adds elements to the beginning of an array.

let arr = [2, 3];
arr.unshift(1);
console.log(arr); // [1, 2, 3]

Immutable Methods

.map()

Creates a new array by applying a function to each element.

let nums = [1, 2, 3];
let squared = nums.map(x => x * x);
console.log(squared); // [1, 4, 9]

.filter()

Creates a new array with elements that pass a test.

let nums = [1, 2, 3];
let even = nums.filter(x => x % 2 === 0);
console.log(even); // [2]

.reduce()

Reduces an array to a single value by applying a function.

let nums = [1, 2, 3];
let sum = nums.reduce((acc, curr) => acc + curr, 0);
console.log(sum); // 6

.slice()

Returns a shallow copy of a portion of an array.

let arr = [1, 2, 3];
let part = arr.slice(1, 3);
console.log(part); // [2, 3]

Other Methods

.find()

Finds the first element that satisfies a condition.

let nums = [1, 2, 3];
let found = nums.find(x => x > 1);
console.log(found); // 2

.findIndex()

Finds the index of the first element that satisfies a condition.

let nums = [1, 2, 3];
let index = nums.findIndex(x => x > 1);
console.log(index); // 1

.includes()

Checks if an array contains a specific value.

let arr = [1, 2, 3];
console.log(arr.includes(2)); // true
